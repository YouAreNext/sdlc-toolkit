description: Create an Architecture Decision Record (ADR)
---

Create an Architecture Decision Record to document a technical decision.

Use the adr skill. Input data: $ARGUMENTS

$ARGUMENTS can be:
- Iteration slug (e.g.: "01-article-processing") — ADR linked to that iteration
- Decision title/description — create standalone ADR
- Empty — ask the user what decision to document

If related to a specific feature, save to docs/discovery/NN-slug/decisions/.
Otherwise save to docs/adr/.
